# PronoucePal

voice-driven english practice app

## setup

frontend:
```
npm install
npm run dev
```

backend:
```
cd backend
npm install
node server.js
```

## env vars

frontend (.env):
- VITE_API_URL
- VITE_ELEVEN_KEY
- VITE_GEMINI_KEY

backend (backend/.env):
- PORT
- GEMINI_API_KEY
- ELEVENLABS_API_KEY
